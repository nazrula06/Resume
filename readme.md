## 🎨 Customization

To customize this resume:

1. Update contact information in the header section
2. Modify colors by changing the hex codes in `styles.css` (currently using purple/blue gradient)
3. Add or remove sections as needed
4. Update all content with your own information

### Color Scheme
- Primary: `#667eea` (Blue)
- Secondary: `#764ba2` (Purple)
- Background: `#f8f9fa` (Light Gray)
